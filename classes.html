Coding Camp Session #1

We will be downloading JDK and jGrasp. We will also introduce Java with Scratch, which is beginner friendly. Introduce primitive types (int, double, boolean, char). Students will understand how to write a basic HelloWorld program. They will also understand the difference in String and other primitive types (Strings... how they are similar to char). They will finally learn escape sequences (\n \t \" \' \\).
